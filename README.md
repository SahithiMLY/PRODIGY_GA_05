# PRODIGY_GA_05

## Neural Style Transfer using TensorFlow

### Objective
Apply the artistic style of one image to another image using Neural Style Transfer with TensorFlow Hub.

---

## Technologies Used

- Python
- TensorFlow
- TensorFlow Hub
- NumPy
- Matplotlib
- Pillow
- Google Colab

---

## Project Workflow

1. Import required libraries.
2. Upload the content and style images.
3. Preprocess the images.
4. Load the pre-trained Neural Style Transfer model from TensorFlow Hub.
5. Apply the artistic style to the content image.
6. Generate the stylized output image.
7. Save the final output.

---

## Input

- Content Image
- Style Image

---

## Output

A stylized image that preserves the content of the original image while applying the artistic style of the selected artwork.

---

## Learning Outcomes

- Image preprocessing
- Deep Learning for image generation
- Neural Style Transfer
- TensorFlow Hub model usage
- Transfer Learning
